# RenewBoost - 面向 TypeRenew 的轻量优化加速插件

本插件是一款面向 TypeRenew 的轻量优化加速插件，适配 PHP 8 与 MySQL 8，围绕头像链路、本地图片 WebP、前台图片改写、输出压缩与响应头协同，提供一套默认可用、便于统一管理的基础加速能力。

## 功能特性

- **头像加速**
  - 支持切换 Gravatar 镜像源
  - 支持全局替换前台评论头像与后台用户头像
  - 支持 QQ 邮箱头像优先获取
  - 支持输出头像域名的 `dns-prefetch` 与 `preconnect`
  - 支持头像结果缓存与失败退避重试

- **本地图片优化**
  - 上传或替换本地图片时自动生成 `.webp` 副本
  - 支持 `Imagick` 或 `GD` 引擎
  - 原图继续保留，不破坏原有附件路径
  - 正文与摘要可自动输出 `<picture>` 结构
  - 支持首图取消延迟加载、其余图片按需懒加载
  - 支持评论图片改写、Feed 改写与路径排除规则

- **输出与头部协同**
  - 支持在条件满足时启用前台 `gzip` 输出压缩
  - 自动规避已启用 `zlib.output_compression` 或现有 gzip 处理器的重复压缩
  - 支持为 HTTPS 站点输出自定义 `Alt-Svc` 响应头

- **缓存协同**
  - 可联动 TypeRenew 已启用的 Redis 或 APCu 缓存
  - 用于保存头像结果与图片改写候选缓存
  - 支持独立配置缓存时长

## 适用场景

- 基于 TypeRenew 构建的个人博客、内容站和轻量网站
- 希望在不引入复杂性能系统的前提下补齐基础加速能力的站点
- 需要统一管理头像、图片与输出链路优化开关的场景
- 已启用系统缓存，希望顺带联动头像与图片缓存的站点

## 安装方式

1. 将插件目录上传到站点的 `/usr/plugins/` 目录
2. 确保目录名称为 `RenewBoost`
3. 登录后台，进入"控制台 -> 插件"
4. 找到并启用 `RenewBoost`
5. 启用后，在左侧导航中进入"优化加速"面板完成配置

## 面板说明

插件后台面板分为以下模块：

- **基础设置**
  - 插件总开关
  - 系统缓存联动
  - 配置缓存秒数

- **头像加速**
  - 镜像源切换
  - QQ 邮箱头像优先策略
  - 全局头像源替换
  - 头像资源提示输出
  - 自定义镜像、缓存与超时配置

- **图片优化**
  - 本地图片 WebP 转换
  - 前台正文与摘要图片改写
  - 图片资源提示输出
  - 图片懒加载与正文首图例外
  - 评论图片改写、Feed 改写与排除路径

- **输出与头部**
  - 前台输出压缩
  - `Alt-Svc` 响应头输出
  - 当前环境可用性说明

## 工作方式

### 头像链路

- 插件可接管评论头像输出，并在需要时定义全局 Gravatar 前缀
- 对 `12345@qq.com` 这一类邮箱可优先尝试获取 QQ 头像
- 若检测到 `RenewAvatar` 已启用，建议停用旧插件后再启用本插件头像功能，避免重复接管

### 图片链路

- 仅处理站点上传目录中的 `jpg`、`jpeg`、`png`、`gif` 文件
- 动图 `gif` 不参与 WebP 转换
- 仅当生成的 `.webp` 体积更小且文件有效时才保留副本
- 前台改写仅作用于插件识别到的本地上传图片，不改写外链图片

### 输出链路

- 输出压缩仅对前台 `GET` 与 `HEAD` 请求生效
- 若请求带有 `Range`，或环境已存在压缩处理器，插件会自动跳过
- `Alt-Svc` 仅负责输出响应头，不负责创建实际的 HTTP/3 服务端点

## 目录结构

```text
RenewBoost/
├── assets/              # 后台面板静态资源
├── view/                # 面板视图模板
├── Plugin.php           # 插件入口与 Hook 注册
├── Settings.php         # 配置默认值、归一化与缓存协同
├── Panel.php            # 后台面板入口
├── Action.php           # 配置保存与面板动作入口
├── Avatar.php           # 头像链路加速
├── Image.php            # 本地图片 WebP 与前台改写
├── Output.php           # 输出压缩与 Alt-Svc 头处理
└── Text.php             # 文本辅助工具
```

## 注意事项

- 本插件提供的是轻量优化能力，不替代 CDN、对象存储、反向代理缓存或专业图像服务
- 图片优化仅针对站点本地上传目录，不负责远程图片抓取或转存
- 启用输出压缩前，建议先确认服务器或上游环境没有重复开启 gzip 压缩
- 若站点已单独使用头像插件，建议避免与 `RenewBoost` 的头像功能同时启用
